🔬 Waypoint — Diagnostic Tool

ServiceNow REST API Tester

Tests whether Waypoint can access the ServiceNow API using your existing browser session — the same technique that powers the Jira Migration Tool. Nothing to install. No API key needed.

💡 How it works

🌐
You're already logged into ServiceNow — your browser holds a live session cookie.
📎
The bookmarklet runs inside your ServiceNow tab — same origin means every fetch() call automatically includes your session cookies.
🔑
No API key required — exactly the same technique used by the Jira Migration Tool to call /rest/api/2/ without credentials.
📊
Results tell us whether to replace DOM scraping with clean, reliable REST API calls in Waypoint.
