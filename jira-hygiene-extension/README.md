# Jira Hygiene Assistant

Browser extension for finding and fixing Jira ticket hygiene issues.

## Installation

1. Open Chrome/Edge
2. Go to chrome://extensions or edge://extensions
3. Enable "Developer mode"
4. Click "Load unpacked"
5. Select this folder

## Usage

1. Click extension icon
2. Enter your Jira URL and save
3. Open any Jira page
4. Click extension icon and run queries
5. Use bulk actions on found tickets

## Features

- Find stale tickets (7+ days no update)
- Find missing descriptions
- Find missing due dates
- Custom JQL queries
- Bulk add comments

Works with Jira Server and Cloud via REST API v2.
