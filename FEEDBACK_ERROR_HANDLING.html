üîç Common Issues & Solutions

"Bad credentials" or "401 Unauthorized"

Cause: Invalid or expired GitHub token

Solution:

  1. Go to github.com/settings/tokens
  2. Create new token with repo or public_repo scope
  3. Copy token immediately (only shown once)
  4. Go to Waypoint Settings ‚Üí Feedback Configuration
  5. Paste token and save

"Repository not found" or "404"

Cause: Wrong repository format or no access

Solution:

  • Format must be: owner/repository
  • Examples: mikejsmith1985/jira-automation
  • NOT: https://github.com/owner/repo ‚ùå
  • NOT: github.com/owner/repo ‚ùå
  • Ensure you have write access to the repository

"Permission denied" or "403 Forbidden"

Cause: Token lacks required permissions

Solution:

  • Token must have repo scope (for private repos)
  • OR public_repo scope (for public repos only)
  • Regenerate token with correct scope at github.com/settings/tokens
